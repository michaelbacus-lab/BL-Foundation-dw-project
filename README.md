# BL-Foundation-dw-project
SQL Server Data Warehouse implementing a Bronze–Silver–Gold architecture with ETL pipelines, dimensional modeling, and business-ready views using synthetic membership management data.

# Data Notice

This repository is based on a real-world Membership Management System developed for a housing and community organization.

To protect the privacy of individuals and comply with data confidentiality requirements:

- No production data is included in this repository.
- All datasets have been anonymized and contain synthetic data.
- Personal identifiers such as names, contact numbers, addresses, and project names have been replaced with fictional values.
- Financial values and dates have been modified where appropriate while preserving the overall data structure and business logic.
- The data models, ETL processes, and data quality rules reflect real business scenarios, but the published datasets are intended solely for educational and portfolio purposes.

The goal of this repository is to demonstrate Data Engineering concepts, including data ingestion, cleansing, transformation, validation, and dimensional modeling, without exposing confidential organizational or customer information.
